Modern marketing site for **Planora**, an AI-assisted travel planner. Built with Next.js (App Router) and Tailwind CSS to showcase a smart, elegant travel-planning experience that can deploy seamlessly to Vercel.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to explore the experience locally. Build with `npm run build` and preview production using `npm run start`.

## Tech Highlights

- Next.js 14 (App Router) with React 18
- Tailwind CSS custom theme with gradients, shadows, and typography tuned for a premium travel brand
- Responsive sections for hero, AI trip planner workflow, destination recommendations, and assistant chat
- Floating AI chat widget with smart suggestions and playful interactions

## Deploy

```bash
npm run build
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-cff1caab
```
